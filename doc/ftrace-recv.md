NAME
====
ftrace-recv - Receive tracing data from socket and save it to files

SYNOPSIS
========
ftrace recv [*options*]

DESCRIPTION
===========
This command receives tracing data from network and saves it to files.

SEE ALSO
========
`ftrace`(1), `ftrace-record`(1)
